## Project Overview

This project simulates CPU scheduling algorithms and deadlock detection in an operating system environment. The following features are implemented:

1. CPU Scheduling Algorithms:
   - First-Come-First-Serve (FCFS)
   - Shortest Job First (SJF)
   - Round Robin (RR) with Time Quantum
2. Deadlock Detection:

3. Input / Output Handling:
   - Input from CSV files
   - Text-based Gantt charts
   - Average Waiting Time and Turnaround Time calculation
